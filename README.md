# LDAPToRadius

A small script to bridge between LDAP and Radius for authentication.

## Dependencies

* Ruby >= 1.9.3
* Rubygems
* Bundler

## Usage

### Start

To start a server:

```text
$> bundle exec rake server:start
```

### Stop

To stop a server, hit `Ctrl-\`.

## Test

To run the tests:

```text
$> bundle exec rake spec
```
